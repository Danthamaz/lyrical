Memory Blur

How memory blur works and how to maximize your blur chance

What is Memory Blur

Memory blur (memblur) wipes an NPC's hate list, causing them to forget you and return to their spawn point. It's one of the most useful tools in a bard's kit for managing encounters. Common uses include:

  • Splitting camps — blur adds off a multi-pull so you only fight one at a time.
  • Dropping aggro — blur a mob that's chasing you to reset the fight.
  • Escort quests — blur escort NPCs to speed them up (though bards have limitations here).

Bard Blur vs Direct Blur

Bards don't get a dedicated blur spell. Instead, blur is a side effect of mesmerize spells — when a mez lands, it includes a blur component. This distinction matters because it comes with several important limitations:

  • Mez must land first — the blur component only fires if the mez itself passes the resist check. If the mez is resisted, no blur happens.
  • Re-mezzing skips blur — if a target is already mezzed and you re-mez it (a refresh), the blur component is skipped entirely. The mez refreshes, but no new blur attempt is made.
  • Mez-immune targets can't be blurred — if an NPC is immune to mesmerize (unmezzable), bards have no way to blur it.
  • Enchanters have direct blur — classes like Enchanter get standalone memory blur spells that don't require mez. These work on targets that are mez-immune and can be cast repeatedly without needing to break mez first.

The Formula

Memory blur isn't a flat chance — it's calculated from several components that stack together. Understanding the formula helps you know when blur is reliable and when it's going to be a coin flip.

Base Chance

Every blur-capable spell has a base value that sets the starting success chance. This varies by spell — higher level mez spells generally have a higher base blur chance.

IncreaseChanceMemwipe Bonus

Bonuses from AAs, items, and spell effects that specifically increase memory wipe chance. This is applied as a percentage multiplier on the spell's base chance:

base += base × bonus / 100

This modifier is applied before the level and CHA modifiers are added.

Level Modifier

The target's level determines a bonus (or penalty) to your blur chance:

  • Target level below 17: Always 100% success — auto-blur, no roll needed.
  • Target level 17–52: Bonus = -2.08333 × target_level + 135.4167 (higher level = smaller bonus = harder to blur).
  • Target level 53+: Flat +25 bonus.

CHA Bonus

Charisma directly impacts blur chance, but only above 150. This is one of the key reasons CHA matters for bards.

  • Only applies if caster has over 150 CHA.
  • Formula: (CHA - 150) / 10, capped at +15.
  • So 300+ CHA gives the maximum +15% bonus.

Final Formula

Putting it all together, the blur chance is calculated in this order:

  1. Start with the spell's base blur chance.
  2. Apply IncreaseChanceMemwipe multiplier: base += base × bonus / 100.
  3. Add the level modifier (based on target level).
  4. Add the CHA bonus (if CHA > 150).
  5. Cap the final value at 100%.
  6. Roll a random number against the final chance — if the roll is under, the blur succeeds.

Immune NPCs

Some NPCs have a special "engage notice" flag that makes them completely immune to memory blur, regardless of your stats or spell power. When you attempt to blur one of these targets, you'll see the message:

"You can't seem to wipe the mind of this target..."

There is no way around this — these NPCs simply cannot be blurred by any class or spell.

Tips

  • Stack CHA above 150 — every 10 CHA above 150 gives +1% blur chance, up to +15% at 300 CHA. This is one of the best reasons for bards to invest in Charisma gear.
  • You can't re-blur a mezzed target — the mez must wear off or be broken first. Then re-mez to get another blur attempt. Spamming mez on an already-mezzed mob does nothing for blur.
  • Escort quests need direct blur — most escort NPCs are immune to mesmerize. Since bard blur requires mez to land, you'll need a class with a direct blur spell (like Enchanter) for escort quest tricks.
  • Higher level targets are harder — the level modifier drops sharply as target level goes up. Expect more failed blurs on raid-level and high-end dungeon mobs.
