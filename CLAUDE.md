# Lyrical.gg — Project Rules

## HTML File Structure

- Use `<!-- #region Name -->` and `<!-- #endregion -->` to wrap all logical sections in HTML files. These collapsible regions make large files navigable in VS Code and other editors.
- Nest regions where appropriate (e.g., a guild region containing recruit/apprentice/journeyman sub-regions).
- Every new section must have a region wrapper — never use bare comments as section dividers.
