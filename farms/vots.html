Voice of the Serpent (VoTS)

Solo farming guide for the +60% singing modifier necklace

Overview

The Voice of the Serpent is a necklace that provides +60% singing modifier. Some people get it in 10 minutes, some in 20+ hours. Be prepared for a long camp. This guide covers how to solo it as a bard.

Video guide: Voice of the Serpent (VotS) — credit to Lethan the Bard Guy.

Setup

Gear & Song Bar Configuration

▶

For mid-level gear (~3400 unbuffed HP), the mob hits hard and resists mez and snare constantly. It is doable without Ancient Lullaby.

OOS (Occlusion of Sound) is a critical debuff for this camp — run it instead of Amplification.

Set up two song quickbars using Zeal: one for drain/dispel, one for mez/melee/regen.

CRITICAL: Keep mez in the same song slot across both bars. If the mob breaks mez while you are swapping songs, you cannot afford to be stuck without mez on your bar.

Having a fungi set makes this camp go roughly 2x as fast. It is worth the camp or the plat.

Phase 1: Draining

Running Mana Drains

▶

Do a swarm circle up top while casting mana drains and watching resists to see when the mob stops casting and is actually out of mana. This usually takes at least 4 minutes of circles.

Song setup: Elemental or Purifying Chorus (elemental for acolytes, purifying for spiritwards), Ervaj's mana drain, Denon's mana drain, dispel. Run on melody while spinning circles.

Phase 2: Meleeing

Melee Melody

▶

Using Assonance is dangerous here — the mob can hit for ~500 in a single double attack, and DoTs would prevent mezzing if things go wrong.

Run Largos instead for the AGI debuff.

Melody: Largos, McVaxius, Warsong, Damage Shield (flex slot), OOS.

Phase 3: Regenerating

Mez Cycle & Regen

▶

Start the mez cycle at ~45% HP.

Do not be surprised if he bashes you, resists 2 mezzes, then stuns you. Deftdance saves lives here.

Regen melody: OOS, Mez, Mez, Cantata, Niv's.

Running OOS first prevents frequent mez resists. Swap drums in for extra safety.

Run both fungi pieces once the mob is mezzed.

Mezzing 2x in a row followed by OOS ensures he is never memblurred for more than 3 seconds.

Breaking mez with a Dirge to try for crits was the only fun part of this camp.
