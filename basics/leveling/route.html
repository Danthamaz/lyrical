Best leveling path for bards on Project Quarm

Overview

Bards have multiple leveling strategies: swarming (AoE PBAoE kiting), charm kiting, fear kiting, and traditional grouping. This guide covers the best zones for each level range with ZEM (Zone Experience Modifier) values.

New to bard? See Character Creation and Zeal & Setup first.
