Zeal Setup

Installing and configuring Zeal for Project Quarm

What is Zeal

Zeal is an open-source quality-of-life modification for the TAKP (The Al'Kabor Project) client used on Project Quarm. It adds modern conveniences like an in-game map, customizable keybinds, nameplate options, and dozens of slash commands that make the 1999-era client far more playable.

Zeal is developed openly on GitHub with transparent releases. Every build is compiled from public source code, so you can inspect exactly what it does. It is the standard addon that virtually every Quarm player uses.

Installation

  1. Download the latest release ZIP from the Zeal GitHub releases page.
  2. Extract the contents of the ZIP into your EverQuest game root directory (the folder containing eqgame.exe).
  3. Make sure zeal.asi is in the game root folder — not in a subfolder.
  4. Launch the game and test the installation:
    • /zeal version — displays the installed Zeal version
    • /help zeal — lists all available Zeal commands

Note: If Zeal does not load, ensure your antivirus or Windows Defender is not blocking or quarantining zeal.asi. Adding your EQ folder to the exclusion list usually fixes this.

File Structure

After installation, your game directory will contain the following Zeal-related files and folders:

File / Folder Purpose
zeal.asi Core Zeal module — must be in the game root
zeal.pdb Debug symbols (optional, used for crash reporting)
uifiles/zeal/ Zeal UI elements (options window, map, nameplates, etc.)
zeal.ini Global Zeal settings (auto-generated on first run)
client.ini Per-character keybinds and UI settings
zeal_ui_skin.ini Optional skin configuration for 4K UI scaling

Essential First Steps

Once Zeal is installed, open the Zeal options window in-game with ALT+O and navigate to the Zeal tab. Here are the key things to configure right away:

Keybinds

Zeal adds new bindable actions under the Keybinds section. Set up binds for commonly used features like target cycling, strafe, and slow turn. Per-character keybinds are saved in client.ini.

Right-Click to Equip

Enable this in Zeal options. Lets you right-click items in your inventory to equip them instantly — essential for bard instrument swapping.

Instrument Swapping

With right-click equip enabled, you can swap instruments mid-melody without opening bags. This is critical for maximizing song modifiers during twisting.

Per-Character Keybinds

Zeal supports per-character keybind profiles stored in client.ini. Each character can have their own binds without overwriting your other characters.

4K Support

Zeal supports native 4K resolution. However, the default EQ UI elements were designed for much lower resolutions and will appear tiny at 4K. To fix this:

  • Install a custom 2x UI skin that doubles the size of all UI elements.
  • Configure the skin path in zeal_ui_skin.ini in your game root directory.
  • Several community-made 4K skins are available on the Quarm Discord.

Note: Without a 2x skin, the UI will be functional but extremely small at 4K. This is a TAKP client limitation, not a Zeal issue.

To Disable

If you ever need to disable Zeal, simply delete zeal.asi from your game root directory. The game will launch without Zeal the next time you start it. Your settings in zeal.ini will be preserved if you reinstall later.
